# y=x^2近似してくれるAI開発  
使用したのは  
- python3.12
ライブラリは最小限で
- numpy
- matplotlib
計算と描画のためのライブラリ以外は絶対に使わず開発を行っていきます。
## 6/27時点  
行ったこと：AIの基礎設計  
入力層->隠れ層(3-neurons)->出力層の2層ニューラルネットワークで構成  
学習方法は最も基礎的な勾配降下法  
活性化関数はtanh()のみ  
## 7/4時点  
行ったこと：動的学習率の導入  
学習率を学習時に動的に変更する処理を行うことでより効率の良い学習をできるようにする狙い  
## 7/14時点  
行ったこと：入出力データの正規化とグラフ描画においてそれらに伴う逆正規化、そして正規化に伴う学習率の変更  
まず、入力データを正規化（今回は[-1, 1, 0.1]）することで入力データのレンジ[-10, 10, 1]と重みのレンジ（~1）を揃えることができ、  
誤差関数が急峻な概形（イメージ）になってしまうのを防ぎ、なだらかな円錐状に近づけることで勾配降下法による無駄なジグザグなどの意図しない進行  
を防ぐことができるようになった。  
この過程でyの値も[0, 100]->[0, 1]に正規化してあげる必要が出てきたため、リスケール。  
（yだけそのままにすると2乗和平均誤差の値が大きくなりすぎて勾配消失を引き起こすため。）  
そしてこの２つを変更したゆえパラメータ更新のときに前述の通り数値のレンジが揃ったため、学習率も約100倍変化させることとなった。  
これらにより今までより明らかにフィッティングが早くなった。
